# Requisitos del Sistema

Para poder ejecutar, desarrollar o colaborar en este proyecto, asegúrate de tener instalado el siguiente software en tu equipo:

## 🟢 Entorno de Desarrollo (Recomendado)

### 1. Node.js
Es el motor principal para ejecutar tanto el cliente como el servidor.
- **Versión mínima:** `18.x.x`
- **Versión recomendada:** `20.x.x` (LTS)
- [Descargar Node.js](https://nodejs.org/)

### 2. npm (Node Package Manager)
Viene instalado por defecto con Node.js. Se utiliza para la gestión de dependencias.
- **Versión mínima:** `9.x.x`

### 3. Git
Necesario para la gestión de versiones y clonar el repositorio.
- [Descargar Git](https://git-scm.com/)

---

## 🔵 Contenerización (Opcional)

Si deseas ejecutar el proyecto sin instalar todas las dependencias de Node localmente, o para asegurar un entorno idéntico al de producción:

### 1. Docker
- Necesario para crear las imágenes de los contenedores.
- [Descargar Docker Desktop](https://www.docker.com/products/docker-desktop/)

### 2. Docker Compose
- Generalmente incluido en Docker Desktop. Permite orquestar el contenedor del cliente y del servidor a la vez.

---

## 🟡 Herramientas Adicionales
- **Visual Studio Code:** El editor de código recomendado.
- **Extensiones de VS Code:**
  - `ESLint`: Para mantener la calidad del código.
  - `Prettier`: Para el formateo automático.
